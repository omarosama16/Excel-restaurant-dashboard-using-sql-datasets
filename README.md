#  Restaurant Sales Dashboard (Excel)

##  Project Overview

This project presents an interactive **restaurant sales dashboard** built using **Microsoft Excel**.
It focuses on analyzing customer behavior, order patterns, and overall performance through clear and dynamic visualizations.

---

##  Objectives

* Understand customer behavior (local vs non-local)
* Identify the most used ordering channels
* Analyze popular menu items and order types
* Evaluate customer ratings and satisfaction
* Present insights in an interactive dashboard

---

##  Tools & Technologies

* **Microsoft Excel** (Pivot Tables, Charts, Slicers, Dashboard Design)

##  Dashboard Features

The dashboard includes the following key visualizations:

*  **Customer Type Distribution** (Local vs Non-Local)
*  **Order Type Distribution** (Dine-In, Delivery, Takeaway)
*  **Most Ordered Item Types**
*  **Order Analysis by Type**
*  **Average Rating by Channel**

---

##  Key Insights

* Customer segmentation highlights behavior differences
* Order types reveal preferred service channels
* Certain items dominate total orders
* Ratings provide insight into service quality across channels
